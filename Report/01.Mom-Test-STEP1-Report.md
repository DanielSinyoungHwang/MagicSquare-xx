## 4. 수집된 과거 사실 (Past Facts)

- 4.1 Magic Square(ECB) 실습을 사진과 같이 진행했고 **어려움이 있었다**.
- 전체 작업(풀이+ECB+검증)에 **약 20분** 소요.
- **대각선 1개 확인을 놓쳤고**, **20분 뒤 마지막** 검증에서야 알게 됨.
- 알게 된 뒤 **고쳐서 맞췄다** (당일 포기 아님).
- **검증 중 주대각선**을 보다가 틀림을 발견.
- 그 전까지 **행·열은 다 맞다**고 생각.
- 행·열 OK·대각선 NG 패턴은 **이번이 처음**.
- 20분 중 **가장 긴 구간은 빈칸 채우기**, 막힘은 **합 34 맞추기**, **손계산만** 반복.

## 5. 표면 문제 vs 진짜 문제

### 표면 문제 (솔루션 섞임 — 금지 구역)

| 발언/아이디어 | 솔루션 키워드 |
|---------------|---------------|
| 4×4 마방진 **자동 풀이 프로그램** 만든다 | 자동 풀이, 프로그램 |
| ECB **MissingFinder·Solver 앱** | 앱, Solver |
| 34 맞추는 **계산기·힌트** | 계산기, 힌트 |
| **검증 자동화 CLI** | 자동 검증, CLI |

### 진짜 문제 (한 문장)

> 빈칸 2개를 채우고 행·열은 맞다고 믿은 뒤, **주대각선 검증을 마지막에야 하다가 틀렸음을 알게 되어** 약 20분짜리 작업 전체를 **늦게 다시 판정·수정**해야 했다.

### Mom Test 증거 (3줄)

1. "사진처럼 ECB 작업을 했는데 **어려움이 있었어**."
2. "**대각선 1개 확인을 놓쳤었고**, **20분 뒤 마지막에** 알았어."
3. "주대각선 확인 전까지 **행·열은 다 맞다고 생각**했고, **검증하다가** 틀렸다는 걸 알았어."
